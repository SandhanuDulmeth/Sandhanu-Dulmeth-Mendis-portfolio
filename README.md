# Windows XP Portfolio

A React-based portfolio application that mimics the classic Windows XP operating system interface. This project brings back the nostalgia of the early 2000s web and OS design, built with modern web technologies.

## Features

- **Boot Sequence**: Authentic Windows XP boot screen animation.
- **Login Screen**: Interactive user login interface.
- **Desktop Environment**: Classic Windows XP desktop layout with icons and taskbar.
- **Window Management**: Functional window system for applications.
- **Themes**: Custom XP styling using `xp-theme.css`.

## Tech Stack

- **Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **State Management**: React Context API (OS State)
- **Styling**: CSS Modules / Vanilla CSS

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```
